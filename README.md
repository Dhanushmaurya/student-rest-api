# Student REST API

This is a simple REST API project built using Node.js, Express.js, and MySQL.

## Features

- Create student data
- Read all students
- Read single student by ID
- Update student information
- Delete student record

## Technologies Used

- Node.js
- Express.js
- MySQL
- Body-parser

## API Endpoints

GET /college_student  
GET /college_student/:id  
POST /college_student  
PUT /college_student/:id  
DELETE /college_student/:id  

## How to Run the Project

1. Install Node.js
2. Install dependencies
